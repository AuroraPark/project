## 3. 로그인 시도

> 메일함에서 인증 성공

```
이메일로 회원가입을 진행해 보도록 하겠습니다.
에이젝스를 활용하여 한글자 입력될 때마다 db에 저장되어있는 이메일과 닉네임을 비교하여 중복검사를 진행하게 됩니다.
선호 장르를 선택하고, 약관을 동의한 후 회원가입 버튼을 클릭합니다.

다시 이메일에서 링크를 클릭하여 인증하게 되면 db에서 메일인증컬럼이 Y로 변경되어 로그인이 됩니당!

원활한 진행을 위해 데이터가 들어있는 아이디로 다시 로그인 하도록하겠습니다.

```
